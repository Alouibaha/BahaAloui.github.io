development of a calculation note application

As part of my master's degree, at INSA Hauts-de-France, I did an internship at LSB La Salle Blanche. LSB proposes to design and install clean rooms and controlled atmosphere areas for the pharmaceutical industry, the hospital industry and also the research.
The objective of my internship was to understand the calculations made for air treatment in clean rooms and develop an application that processes them. To build this application I used Dash, a Python framework, which allows to create web applications for data analysis, visualization, and modeling.
The web application built establishes an energy and psychrometric balance for new air handling systems. Below are some screenshots of the calculations made by the application. This version of the application is a simpler model to guarantee the confidentiality of the project.
